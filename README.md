# IPL Analytics Dashboard

Interactive IPL Analytics Dashboard built using Python, Pandas, Plotly, and Streamlit.

## Features
- Team Performance Analysis
- Player Performance Analysis
- Toss Analysis
- Venue Analysis
- Interactive Filters
- KPI Cards

## Tools Used
- Python
- Pandas
- Plotly
- Streamlit

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
